El número de etapas que tienen los proyectos de esta modalidad.

La fecha de inicio de registro de solicitudes para esta modalidad. Expresado con ISO 8601. https://www.iso.org/iso-8601-date-and-time-format.html
public fechaFin String

La fecha fin de registro de solicitudes para esta modalidad. Expresado con ISO 8601. https://www.iso.org/iso-8601-date-and-time-format.html
